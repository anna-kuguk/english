# English Daily

Персональное приложение Маши для системного улучшения английского: цель Band 7.0 на следующем IELTS через ежедневные 20-30 минут.

**Живая версия:** https://anna-kuguk.github.io/english/

## Что внутри

- **Сегодня** - план дня из 6 блоков (повторение, новые слова, текст, грамматика, письмо, речь). От 4 закрытых блоков день идёт в серию (streak), 6 из 6 - «идеальный день».
- **Слова** - 270+ слов и выражений (7 базовых тем + 10 бизнес-тем + фразовые глаголы, идиомы, разговорные, академические). Интервальные повторения по алгоритму SM-2 (как в Anki): Снова / Трудно / Хорошо / Легко. Три формата повторения: карточка, выбор перевода, пропуск в примере.
- **Чтение** - 10 текстов с таймером (считается скорость чтения, слов/мин), вопросами на понимание и разбором лексики. Слова из текста добавляются в повторения одной кнопкой.
- **Грамматика** - 7 тем: мини-теория + 63 упражнения с мгновенным разбором ошибок.
- **Письмо** - 36 мини-упражнений 6 типов (перефразирование, замена на естественную лексику, соединение предложений, исправление ошибок, дописывание абзаца, короткий ответ) с чек-листом и образцами.
- **Речь** - вопрос дня с записью голоса: транскрипт, темп (слов/мин), слова-паразиты, использование фраз темы. Дриль произношения слов дня. 8 разговорных тем с фразами и образцами ответов.
- **Прогресс** - streak, график 4 недель, статистика по всем модулям, путь к цели.

Прогресс хранится в браузере (localStorage). В настройках (⚙) есть экспорт/импорт - можно перенести прогресс на другое устройство.

## Как установить на телефон

Открой сайт в браузере телефона → меню → **«Добавить на главный экран»**. Приложение работает офлайн после первого открытия.

## Напоминания

Три уровня (рекомендуемый - первый):

1. **📅 Календарь телефона (лучший: без интернета и VPN)** - настройки (⚙) → выбери время → «В календарь». Скачается файл `english-daily.ics`, открой его - календарь будет напоминать каждый день в выбранное время. Работает офлайн, в самолёте, без VPN. Удалить - удалить событие из календаря.
2. **В браузере** - настройки (⚙) → «В браузере». Срабатывает, только пока приложение открыто.
3. **В Telegram (опционально; в РФ Telegram может требовать VPN)** - ежедневно в 19:00 МСК через GitHub Actions:
   1. Создай бота у [@BotFather](https://t.me/BotFather) (`/newbot`) и скопируй токен.
   2. Маша пишет боту `/start`, затем открой `https://api.telegram.org/bot<ТОКЕН>/getUpdates` и найди её `chat.id`.
   3. В репозитории: Settings → Secrets and variables → Actions → добавь `TG_BOT_TOKEN` и `TG_CHAT_ID`.
   Время меняется в `.github/workflows/reminder.yml` (cron в UTC: 16:00 UTC = 19:00 МСК).

## Офлайн

После первого открытия приложение полностью работает без интернета (PWA-кэш): слова, повторения, тексты, грамматика, письмо, план дня, статистика. Прогресс хранится на устройстве.

Не работает офлайн: распознавание речи (транскрипт и проверка произношения - браузер отправляет звук на сервер распознавания). Озвучка слов зависит от голосов в системе: на Android офлайн-озвучка есть, если скачан английский голос (Настройки → Синтез речи).

## Как добавлять контент

Весь контент - обычные JS-файлы в `data/`, по образцу существующих записей:

- `words.js`, `words-biz.js`, `words-lexis.js` - слова (id должен быть уникальным)
- `reading.js` - тексты (wcount считается автоматически)
- `grammar.js` - темы и упражнения (opts+a - выбор, accepted - ввод)
- `writing.js` - письменные задания (must - чек-лист regex, model - образцы)
- `speaking.js` - вопросы дня и темы

После правок: увеличь версию кэша в `sw.js` (`eng-v1` → `eng-v2`) и запушь в `main` - GitHub Pages обновится сам.

## Ограничения

- Распознавание речи (Web Speech API) лучше всего работает в Chrome (Android/десктоп). В Safari поддержка частичная; запись и прослушивание себя работают везде, где есть микрофон.
- «Проверка произношения» - честная эвристика: слово сверяется с тем, что распознал браузер. Это не фонетический анализ, но кривое произношение оно ловит.
